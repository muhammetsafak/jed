# JED <Java Encoder/Decoder>


## Requirements

- Java 8

## Build and Run

```
javac -d ./out/artifacts/jed_jar ./src/JavaEncoderDecoder.java
jar cfe jed.jar JavaEncoderDecoder -C ./out/artifacts/jed_jar .
```

```
java -jar ./out/artifacts/jed_jar/jed.jar
```
